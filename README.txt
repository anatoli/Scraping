    crawler/index.js - запускать до отображения ui. Используется для сборки ID, name -  брендов автомобилей.
    И получает список моделей по брендам. ( поскольку в запросе API используется ID бренда и модели.)

    crawler/module_detail.js - отрабатывает парсер по запросам с параметрами.

    ДАнные сохраняются в av/av_id.json и av/av_detail.json соответственно.
